# **App Name**: DarkWrite

## Core Features:

- User Authentication: Secure user registration, login, and profile management for writers.
- Rich Text Editor: An intuitive writing environment with basic rich text formatting capabilities for composing stories and chapters.
- Story & Chapter Organization: Allows writers to create, rename, and manage stories and their respective chapters, accessible via a left sidebar navigation.
- Real-time Auto-save: Automatically saves written content as the user types, ensuring no progress is lost.
- AI Writing Companion: An AI tool that provides stylistic suggestions, checks for grammar and syntax, or offers creative writing prompts to assist writers.

## Style Guidelines:

- Primary color: Muted Indigo (#5C3D99) for interactive elements and highlights, suggesting depth and creativity in a focused writing environment.
- Background color: Deep Slate Grey (#24202B) with a subtle purplish undertone, providing a calm, dark theme that aids concentration for 'DarkWrite'.
- Accent color: Vibrant Sky Blue (#62A9F7) used sparingly for strong calls to action, active states, and to provide a refreshing contrast.
- Headline and UI font: 'Inter' (sans-serif) for a modern, clean, and objective look that ensures readability across interface elements.
- Body font: 'Alegreya' (serif) chosen for its elegant, intellectual, and contemporary feel, making long-form reading and writing comfortable and engaging.
- Minimalist and clean line icons to maintain an uncluttered and focused writing interface, subtly hinting at their functions.
- A classic three-column layout featuring a narrow left sidebar for navigation and story organization, a main content area for the writing editor, and an optional right-hand panel for writing aids or settings, prioritizing writing focus.
- Subtle, fluid transitions and micro-interactions for saving, loading, and navigating between sections, designed to be unobtrusive and enhance user flow without distraction.
